# 44교시 생존수업 · 크리에이티브 서칭 대시보드

스릴러·공포 카테고리 화제작들의 **광고 크리에이티브를 벤치마크**하여, 재광고 예정인 자사 웹툰 **&lt;44교시 생존수업&gt;** 의 소재 전략 인사이트를 도출하는 단일 페이지 대시보드.

## 🔗 공개 URL (외부 공유 가능)
배포 후 이 자리에 URL — 로그인 없이 누구나 열람 가능.

로컬에서 보려면: `index.html` 더블클릭, 또는 `node server.js` → http://localhost:8849

## 대시보드 구성
1. **벤치마크 작품 개요** — 자사 1 + 화제작 5, 장르·화제성·Meta 광고 집행 현황
2. **비교 분석** — 44교시 vs 요청 작품 크리에이티브 격차 (아젠다 1)
3. **재광고 적용 인사이트** — 44교시 재집행 위닝 전략 7종 (아젠다 2)
4. **후킹 패턴 분석** — 벤치마크 카피·연출 후킹 분류
5. **포맷·매체 분포**
6. **크리에이티브 갤러리** — 실제 카피·키비주얼·후킹 (출처 링크·정확도 등급)
7. **공개·캠페인 타임라인**

## 핵심 발견
- **44교시는 Meta 광고 라이브러리 직접 집행 0건** → 외부 유료광고 미집행. 30.8만 관심수 IP인 만큼 재광고 시 신규 유입 기회 최대.
- 벤치마크 공통 위닝 코드: **일상공간 전복**(학교·회사·가구점), **반전·역설 카피**, **멀티포맷(영상+캐릭터+티저)**, **화제성 수치 전면화**.

## 파일 구조
| 경로 | 역할 |
|------|------|
| `index.html` | 대시보드 화면 (SVG 자체 차트, 네트워크 무의존) |
| `data.js` | 데이터(`window.DATA`) — 작품·크리에이티브·비교·인사이트 |
| `assets/*.jpg` | 실제 키비주얼·예고편 썸네일 (네이버/YouTube 원본) |
| `server.js` | 로컬 미리보기 서버 (의존성 없음) |
| `vercel.json`, `.vercelignore` | 배포 설정 |

## 데이터 정확도
- **카피·키비주얼·메타데이터** = 정확 (공식 플랫폼·언론 원본)
- **화제성 일부 수치·해외 매출** = 추정
- **박스오피스·기사 보강** = 수동
- 출처 URL을 각 크리에이티브 카드에 링크. 수집일: 2026-06-16.

## 내 버전으로 수정/배포 (누구나)
```bash
git clone https://github.com/performanceteam7-rgb/creative-search-44.git
cd creative-search-44
node server.js          # http://localhost:8849 (또는 index.html 더블클릭)
npx vercel --prod       # 내 Vercel로 배포
```
화면은 `index.html`, 데이터는 `data.js`만 수정하면 됩니다.
